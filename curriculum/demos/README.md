# Demo Scripts

## Overview

Interactive demos and investor presentation scripts that showcase COSURVIVAL capabilities.

---

## Demos

### Core Demos
- **`hello_world_demo.py`** - Three-lens interpretation demo
  - Shows how one activity row is interpreted through TRIBE/TEACHER/RECON lenses
  - Demonstrates the core concept of multiple perspectives

- **`pipeline_demo.py`** - 4-stage governance pipeline demo
  - Ingest → Normalize → Govern → Synthesize
  - Shows complete data flow with governance gates

- **`extractors_demo.py`** - MVP extractors demo
  - TRIBE: Communities, bridges, mentors
  - TEACHER: Learning paths, skill gaps
  - RECON: Provider scores, value flows

### Presentation
- **`INVESTOR_DEMO_SCRIPT.md`** - Investor demonstration script
  - Complete presentation flow
  - Key talking points
  - Demo sequence

---

## Running Demos

### Hello World Demo
```bash
python curriculum/demos/hello_world_demo.py
```
Shows three-lens interpretation of a single activity.

### Pipeline Demo
```bash
python curriculum/demos/pipeline_demo.py
```
Demonstrates complete governance pipeline.

### Extractors Demo
```bash
python curriculum/demos/extractors_demo.py
```
Shows MVP extraction for all three systems.
